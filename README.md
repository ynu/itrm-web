# IT资源管理系统

## 概述

使用此系统可以更方便地管理学校内部的各类IT资源（包括网站、应用系统等）。

## 特别注意事项

- 系统必须使用较高版本的谷歌或火狐浏览器才能打开。其他浏览器或低版本浏览器会导致页面不完整、空白或按钮点击无反应等问题；
- 只有学校在编在岗教职工才能使用本系统；
- 本系统需要通过学校统一身份认证系统登录后才能使用；
